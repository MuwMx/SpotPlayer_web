## ✨ Features

### 🎨 Immersive UI/UX
- **Dual Visual Styles** — Toggle seamlessly between a clean, minimalist Gradient theme and an immersive dynamic Blur style.
- **Artwork Color Extraction** — Interface background colors automatically react and blend with the current track's cover art.
- **Premium Text Effects** — Smart edge-fading scrolling layout (`Marquee`) for long titles, preventing text clipping.
- **Fluid Micro-Interactions** — Spring-animated controls, physics-based gesture handles, and smooth panel transitions.

### 🎵 Enhanced Playback
- **Flawless Background Engine** — Keeps the music streaming smoothly with zero interruptions when the app is minimized or the screen is locked.
- **MediaSession Integration** — Native control via the Android system notification drawer, dynamic progress tracking, and lock screen widgets.
- **Reactive Ad-Skipping** — Built-in background watchdog that instantly detects and automatically skips audio advertisements on the fly.

### 🎤 Advanced Lyrics Engine
- **YT Music Style Tracking** — Real-time synchronized lyrics with smooth vertical auto-scrolling and contextual focus/blur animations.
- **Concurrent API Fetching** — High-speed parallel network strategies with intelligent full-text scoring to locate hard-to-find metadata.
- **Fail-Safe Caching** — Intelligent two-layer memory (`LruCache`) and local disk cache management with silent legacy data migration.
- **Click-to-Seek Navigation** — Instantly jump to any part of the song by simply tapping on the corresponding text line.
